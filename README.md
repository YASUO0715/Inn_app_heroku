
# 宿検索投稿App

## About this app
---
* 八幡平の宿をCRUDできるアプリ
* 八幡平の宿を検索できるアプリ

## 実装機能
---
* 現在地から近い順にソート
  * このアプリの核となる機能です。現在地から近い順にソートで表示されGoogleマップ上にピンがドロップされます。

* CRUD
  * 宿がCRUDできるようになっています

* 検索機能
  * 地域での検索が可能です。(今回は八幡平市のみ)

* ファイルアップロード
   * 複数枚画像アップロードへ対応

* Articleモデルに、User Category Status 各モデルが外部キーとして紐付いています。

* SNS認証機能を2つ追加しました。
