
# 宿検索投稿App

## About this app
---
* 八幡平の宿をCRUDできるアプリ
* 八幡平の宿を検索できるアプリ

## 実装機能
---

* CRUD
  * 宿がCRUDできるようになっています

* 検索機能
  * 地域、宿名での検索が可能となっております。(受け入れ状況での検索は今回間に合いませんでした。)

* ファイルアップロード
   * 複数枚画像アップロードへ対応

* Articleモデルに、User Category Status 各モデルが外部キーとして紐付いています。

* SNS認証機能を3つ追加しました。


テーブル設計を失敗したため、住所、電話番号、地図等基本的な情報の追加は間に合いませんでした。
